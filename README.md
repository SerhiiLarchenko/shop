## Getting started

#### Requirements
- Node.js
- npm

#### Run
- In project folder install node packages:
  `npm install`
- Run the API server:
  `npm run api/server`
  * Server will be running on  `http://localhost:8000`
- Open new terminal tab and start react app:
  `npm start`
  * The app will start automatically in browser on `http://localhost:3000`
